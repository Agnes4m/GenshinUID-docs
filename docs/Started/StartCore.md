# 手动启动Core<Badge type="tip" text="简单" />

::: warning

**推荐使用本方法启动Core**

**不推荐使用Bot命令启动Core**

`pdm`启动和`poetry`启动二选一即可

:::

## Poetry启动

- 移动至`gsuid_core/gsuid_core`文件夹内

- 输入命令`poetry run core`或者`poetry run python core.py `


```sh
poetry run core
```

## Pdm启动

- 移动至`gsuid_core/gsuid_core`文件夹内
- 输入命令`pdm run core`或者`pdm run python core.py `

```sh
pdm run core
```
